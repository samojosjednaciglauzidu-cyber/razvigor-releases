# Razvigor Releases

This repo hosts **only build artifacts** for [Razvigor](https://razvigor.com) — the Windows installer `.exe` (GitHub Releases) and the WinSparkle update feed (`appcast.xml`).

The application source code is private. This repo exists so the in-app update checker and installer downloads can be fetched anonymously (GitHub raw/release URLs require a public repo).

Release process is documented in `docs/release-process.md` in the main `razvigor` repo.
